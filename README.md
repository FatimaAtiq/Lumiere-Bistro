<h1 align="center">Lumière Bistro</h1>
<h3 align="center">Restaurant Website using HTML, CSS and JavaScript</h3>
<p>A modern, responsive, and interactive restaurant website built with HTML, CSS, and JavaScript. Lumière Bistro’s website showcases the restaurant’s story, menu, chef, reservation system, and contact information with smooth animations and dynamic features.</p>
<h3>Project Overview</h3>
<p>The Lumière Bistro website provides an elegant and immersive experience for visitors.</p>
<p>Users can:</p>
<ul>
  <li>Learn about the restaurant and its story.</li>
  <li>Browse the menu with category filters.</li>
  <li>Meet the chef.</li>
  <li>Make reservations through a simple form.</li>
  <li>Contact the restaurant.</li>
</ul>
<p>The website includes a scroll progress bar, dynamic navbar styling, smooth scrolling, and responsive card layouts for menu items.</p>
<h3>Features</h3>
<ul>
  <li><b>Responsive Navbar: </b>Changes style when the user scrolls.</li>
  <li><b>Scroll Progress Bar: </b>Indicates the scroll position visually at the top.</li>
  <li><b>Smooth Scrolling: </b>Navigation links scroll smoothly to sections.</li>
  <li><b>Menu Filtering: </b>Users can filter menu items by category (Starters, Main, Desserts, Drinks).</li>
  <li><b>Hover Effects: </b>Cards lift slightly on hover for a modern feel.</li>
  <li><b>Reservation Form: </b>Simple reservation form with submission alert and form reset.</li>
  <li><b>Sections: </b>Hero, About, Menu, Chef, Reservation, Contact, Footer.</li>
</ul>
<h3>Technologies Used</h3>
<ul>
  <li><b>HTML5</b> – Semantic structure for content</li>
  <li><b>CSS3</b> – Styling, grid layout, flexbox, transitions, hover effects</li>
  <li><b>JavaScript</b> – DOM manipulation, smooth scroll, form handling, menu filtering</li>
  <li><b>Google Fonts</b> – Poppins and Playfair Display for typography</li>
  <li><b>Images</b> – High-quality images for hero, menu, chef, and other sections</li>
</ul>
